◆ endElement()

void XMLInterp4Config::endElement ( const XMLCh *const  pURI,
const XMLCh *const  pLocalName,
const XMLCh *const  pQName 
)
overridevirtual

Metoda zostaje wywołana w momencie zakończenia danego elementu XML, np. jeżeli w pliku jest wpis: 

  <Lib Name="Rotate">
  </Lib>

to metoda ta zostanie wywołana po napotkaniu znacznika: 

</Lib>

Jeżeli element XML ma formę skróconą, tzn. 

  <Parametr Name="Rotate"/>

to wówczas wywołana metoda wywołana zostanie w momencie napotkania sekwencji "/>"

Parametry
[in]pURI- (Uniform Resource Identifier) jeżeli nie jest on wyspecyfikowany poprzez użycie atrybutów xmlns (o ile jego użycie jest przewidziane w gramatyce, w tym przykładzie nie jest to przewidziane), to będzie to napis pusty.
[in]pLocalName- umożliwia dostęp do nazwy elementu XML. W podanym przykładzie nazwą elementu XML jest "Lib".
[in]pQName- pełna kwalifikowana nazwa. To ma znaczenie, gdy użyta jest przestrzeń nazwa. Wówczas poprzez ten parametr można otrzymać nazwę elementu wraz z prefiksem przestrzeni nazwa. Jeśli przestrzeń nazw nie jest użyta (taka jak w tym przykładzie), to pQName i pLocalName dostaczają identyczne napisy.
